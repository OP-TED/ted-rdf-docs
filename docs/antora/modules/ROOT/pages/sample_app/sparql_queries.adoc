In the context of the TED-SWS, SPARQL queries can be used to query  notices data published in RDF by the Publications Office.  These queries can be made on the
https://publications.europa.eu/webapi/rdf/sparql[SPARQL end point] available on EU Vocabularies.

SPARQL stands for "SPARQL Protocol and RDF Query Language". SPARQL queries retrieve and manipulate data stored in RDF (Resource Description Framework) format.  It allows users to write complex queries to retrieve specific information from RDF datasets.

The data available covers Contract Award Notices published from the 28/08/2023 onwards. Data on eForm Result Notices are not yet available.

A query is submitted to the https://publications.europa.eu/webapi/rdf/sparql[SPARQL end point] by replacing the text highlighted below in blue in the Query Text box below with the desired query:

.End point user interface
image::sparql.png[SPARQL End Point]

The format of the result can be changed using the drop-down box of the Results format.

Once the query is in place, and the Results format chosen, the query can be run by pressing “Run Query”.


== Examples
////
=== Query : Find a notice using the (known) notice ID

This query returns all data found relating to a specific notice ID. From this data, the results can be filtered and manipulated further.

TIP: The field that can be changed in this query is the notice ID.

=== Query Code
[,sparql]
----
SELECT * WHERE { graph <http://data.europa.eu/a4g/resource/2023/536107_2023> {?s ?p ?o } }
----

=== Query 1: How many notices have been published by a country on a given date (count only)?

This query returns the count of contract winners by country on a specified date. The data returned are the country name, and the number of contract winners. The data is ordered by winner count descending.



TIP: Fields that can be changed in this query are: the fields selected under "SELECT", and the date.

=== Query Code
[,sparql]
----
PREFIX epo: <http://data.europa.eu/a4g/ontology#>
PREFIX cccev: <http://data.europa.eu/m8g/>

SELECT
?Date
?Buyer_Country
?BuyerLegalNameTrim
COUNT ( DISTINCT ?OJSNumber) as ?NumberOfNotices

WHERE {

VALUES (?Date) {("20230905")}

?Notice epo:hasPublicationDate ?Date .
?Notice epo:hasID/epo:hasIdentifierValue ?OJSNumber.

?Notice epo:announcesRole ?Agent .
?Agent a epo:Buyer.
?Agent epo:playedBy ?Organization .
?Organization owl:sameAs ?SameAsOrganization.

?SameAsOrganization epo:hasLegalName ?BuyerLegalName.
?SameAsOrganization cccev:registeredAddress / epo:hasCountryCode ?BuyerCountryCode .

bind (replace(str(?BuyerLegalName),"(^.*[@])(.*)$","$1") as ?BuyerLegalNameTrim)
bind (replace(str(?BuyerCountryCode),"http://publications.europa.eu/resource/authority/country/","") as ?Buyer_Country)

}
ORDER BY DESC(?NumberOfNotices)
----
=== Data  Returned

NOTE: Some cleaning up of the data is necessary - the bind (replace(str etc statements - to return only the information requested.

=== Query 2: Which is the highest value published for a contract award for a given date by currency (full data)?

This query returns data on the values of published notices on a specific day in descending order of value according to the field, "Lot_AwardedValue".

TIP: Fields that can be changed in this query are the those under the "SELECT DISTINCT" section, and the DATE section.

=== Query Code
[,sparql]
----
PREFIX epo: <http://data.europa.eu/a4g/ontology#>
PREFIX cccev: <http://data.europa.eu/m8g/>

SELECT DISTINCT
?Date
?OJS_and_Lot_Number
?Winner_Name
?Winner_Country
?Lot_AwardedValue
?Currency

WHERE {

 VALUES (?Date ) {("20230905")}

?Notice epo:hasPublicationDate ?Date ; epo:hasID/epo:hasIdentifierValue ?OJSNumber ; epo:announcesContract/epo:includesLot ?Lot .
?LotAwardOutcome epo:describesLot ?Lot ; epo:comprisesTenderAwardOutcome/epo:indicatesAwardOfLotToWinner ?Winner; epo:hasAwardedValue ?LotAwardedValueURI .

?Winner epo:playedBy ?Organization .
?Organization owl:sameAs ?SameAsOrganization.
?SameAsOrganization epo:hasLegalName ?WinnerLegalName. bind (replace(str(?WinnerLegalName),"(^.*[@])(.*)$","$1") as ?WinnerLegalNameTrim).
?SameAsOrganization cccev:registeredAddress / epo:hasCountryCode ?WinnerCountryCode .

?LotAwardedValueURI epo:hasAmountValue ?LotAwardedValue ; epo:hasCurrency ?LotAwardedValueCurrency.

bind( FLOOR(?LotAwardedValue) as ?Lot_AwardedValue).
bind (replace(str(?Lot),"http://data.europa.eu/a4g/resource/id_","") as ?OJS_and_Lot_Number)
bind (replace(str(?WinnerLegalNameTrim),"Da es sich um eine Rahmenvereinbarung mit mehreren Unternehmen handelt, erfolgt gemäß Art 50 Abs 4 Richtlinie 2014/24/EU ","") as ?Winner_Name)
bind (replace(str(?WinnerCountryCode),"http://publications.europa.eu/resource/authority/country/","") as ?Winner_Country)
bind (replace(str(?LotAwardedValueCurrency),"http://publications.europa.eu/resource/authority/currency/","") as ?Currency)


}
ORDER BY DESC(?LotAwardedValueCurrency), DESC(?LotAwardedValue)
----
=== Data  Returned
(To enlarge the image, right-click on it and choose "Open image in new tab")

image::sparql/sparql_answer_new3.png[]

NOTE: The values shown are only those at the beginning of a much longer list in different currencies (in alphabetic order). The others e.g., euros, follow. To get comparable values, all currencies will need to be converted to euros (or the same currency) manually.

=== Query 3: How many awarded contracts foresee subcontracting for a given date (count only)?

This query returns the count of how many awarded contracts foresee subcontracting on the specified date. A simple count is returned.

TIP: The fields that can be changed in this query is the date.

=== Query Code
[,sparql]
----
PREFIX epo: <http://data.europa.eu/a4g/ontology#>

SELECT COUNT(DISTINCT ?Tender) as ?NrOfTenders

WHERE {

   VALUES (?Date ) {("20230905")}

   ?Notice epo:hasPublicationDate ?Date .
   ?Notice epo:refersToLot ?Lot .
   ?Tender epo:isSubmittedForLot ?Lot .
   ?Tender epo:foreseesSubcontracting ?Subcontrating .
}
----
=== Data  Returned
image::sparql/sparql_answer_4c.png[]

NOTE: This count can be compared to the total contract count - see previous queries.

=== Query 4: How many contractors are SMEs by country for a given date (count only)?

This query returns the count of the winning contractors that are SMEs on a specified date. The data is displayed per country, by count descending.

TIP: Fields that can be changed in this query are those under the "SELECT" section and the date.

=== Query Code

[,sparql]
----
PREFIX epo: <http://data.europa.eu/a4g/ontology#>
PREFIX cccev: <http://data.europa.eu/m8g/>

SELECT
?WinnerCountry

COUNT( DISTINCT ?WinnerLegalNameTrim) as ?NrOfWinners

WHERE {

VALUES (?Date ?BusinessSize) {
(
"20230905"
<http://publications.europa.eu/resource/authority/economic-operator-size/sme>
)
}

?Notice epo:hasPublicationDate ?Date .
?Notice epo:hasID/epo:hasIdentifierValue ?OJSNumber.

?Notice epo:announcesRole ?Agent .
?Agent a epo:Winner.
?Agent epo:playedBy ?Organization .
?Organization owl:sameAs ?SameAsOrganization.

?SameAsOrganization epo:hasBusinessSize ?BusinessSize .
?SameAsOrganization epo:hasLegalName ?WinnerLegalName.
?SameAsOrganization cccev:registeredAddress / epo:hasCountryCode ?WinnerCountryCode .

bind (replace(str(?WinnerLegalName),"(^.*[@])(.*)$","$1") as ?WinnerLegalNameTrim)
bind (replace(str(?WinnerCountryCode),"http://publications.europa.eu/resource/authority/country/","") as ?WinnerCountry)

}
ORDER BY DESC (?NrOfWinners)
----

=== Query 5: How many contractors are SMEs by country for a given date (full data)?

This query returns the count of the winning contractors that are SMEs on a specified date. All the data is returned which can then be manipulated manually.

TIP: Fields that can be changed in this query are those under the "SELECT" section and the date.

=== Query Code

[,sparql]
----
PREFIX epo: <http://data.europa.eu/a4g/ontology#>
PREFIX cccev: <http://data.europa.eu/m8g/>

SELECT DISTINCT
?Date
?OJSNumber
?Winner_Name
?Business_Size
?WinnerCountry


WHERE {

VALUES (?Date ) {("20231106")}

?Notice epo:hasPublicationDate ?Date .
?Notice epo:hasID/epo:hasIdentifierValue ?OJSNumber.

?Notice epo:announcesRole ?Agent .
?Agent a epo:Winner.
?Agent epo:playedBy ?Organization .
?Organization owl:sameAs ?SameAsOrganization.

optional {?SameAsOrganization epo:hasBusinessSize ?BusinessSize .}
?SameAsOrganization epo:hasLegalName ?WinnerLegalName.
?SameAsOrganization cccev:registeredAddress / epo:hasCountryCode ?WinnerCountryCode .

bind (replace(str(?WinnerLegalName),"(^.*[@])(.*)$","$1") as ?WinnerLegalNameTrim)
bind (replace(str(?WinnerLegalNameTrim),"Da es sich um eine Rahmenvereinbarung mit mehreren Unternehmen handelt, erfolgt gemäß Art 50 Abs 4 Richtlinie 2014/24/EU ","") as ?Winner_Name)
bind (replace(str(?WinnerCountryCode),"http://publications.europa.eu/resource/authority/country/","") as ?WinnerCountry)
bind (replace(str(?BusinessSize),"http://publications.europa.eu/resource/authority/economic-operator-size/","") as ?Business_Size)
FILTER (?Business_Size = "sme" )
}
ORDER BY ASC (?OJSNumber)


----
