= Query Library
:doctitle: Query Library
:doccode: ODS-SAMP-01

The Query Library is a collection of example SPARQL queries that demonstrate how to retrieve public procurement data from the TED Open Data Service.

== Browse and run queries

All queries are available in the https://data.ted.europa.eu/[TED Open Data Service] under the *Query Library* tab. You can browse the examples by category, preview the SPARQL code, and run any query directly from the editor.

The queries are organised by category and cover topics such as notices, tenders, procedures, organisations, and more.

== Contributing

We welcome contributions to the Query Library. You can help by:

* Adding comments to existing queries to make them easier to understand
* Fixing or improving existing queries
* Contributing new queries that demonstrate useful data retrieval patterns

To contribute, create a branch from `develop` in the https://github.com/OP-TED/ted-rdf-docs[ted-rdf-docs] repository and submit a pull request. For detailed instructions on file format, writing conventions, and how to submit your queries, see the xref:samples:contributing.adoc[Query Library contributing guide].
