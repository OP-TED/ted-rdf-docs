Procurement data in *Cellar* can be queried with Microsoft Excel. A sample Excel file is provided online for download that can be used to paste a query into, which then connects to the endpoint, runs the query, and returns the results back into the Excel file.

////
There are different ways to access TED notices in CELLAR 
and use the data. The methods described below work with TED notices and other type of semantic assets.
////

== Using the Microsoft Excel sample file 

The Excel file can be downloaded https://github.com/OP-TED/ted-rdf-docs/blob/main/spreadsheets/sample_application.xlsm[here].

In the file are the following sheets:

** Query - sheet for pasting into or creating the SPARQL query
** Result Query - a sheet containing the results returned by the query
** Sample analysis -  a sheet contains prefigured dashboard/ charts for the analysis of results returned to the Result Query sheet
